Best Fit Interpolator

A QGIS plugin for selecting, validating, and applying spatial interpolation methods for environmental, soil, and precision-agriculture data.

Best Fit Interpolator compares deterministic, geostatistical, machine-learning, and hybrid interpolation approaches, including IDW, TPS, Ordinary Kriging, REML-assisted kriging, Random Forest, SVM, and Regression Kriging.

Main features

  • Data diagnostics and spatial-pattern support.
  • Semivariogram preview and kriging tools.
  • Framework-guided method selection.
  • Cross-validation metrics and observed-vs-predicted plots.
  • Interpolation maps and PDF report support.
